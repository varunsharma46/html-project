Welcome to CSS!

CSS stands for Cascading Style Sheets. It is a stylesheet language that is used to design and give special properties to the webpage to make it attractive to the user.

It allows us to apply styles to the web pages independent of the HTML.

CSS are of three types as follows:
1. Inline - Inline CSS contains the property inside the body section itself.
2. Internal or Embedded - CSS ruleset is within the HTML file in the head section.
3. External - External CSS contains a seperate CSS file for styling property with the help of tag attributes.
